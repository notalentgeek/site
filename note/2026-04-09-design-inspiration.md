# Design Inspiration

## Discovery Source

Awwwards Nominees — https://www.awwwards.com/websites/nominees/

## Design Principle

"Pragmatic and mindful" — inspired by Indonesian-Japanese style of layouting and physical media (books, pamphlets). Emphasis on intentional spatial organization, editorial flow, and print-influenced design sensibility.

## Primary Inspiration

### OGUD / URBANEX NEXT21 — https://ogud.co.jp/urbanex/next21/

Japanese architectural/housing project site. Key design qualities:

- **Vertical scrolling narrative** — editorial, magazine-like section flow (About, Plan, Voice, Future, Location, Contact)
- **Bold contemporary typography** — Japanese-English bilingual text integration
- **Generous whitespace** — clean spatial organization guiding the eye through content
- **Interactive visual storytelling** — image sliders, before/after toggles, carousel galleries
- **Neutral palette with restrained accents** — lets content and photography lead
- **Print-influenced structure** — sections feel like pages of a well-designed pamphlet

This site is the **main inspiration** for the portfolio direction.

## Secondary Inspirations

### Peter Oravec — https://peteroravec.com/

- **Retro pixel art aesthetic** meets modern web tech — portfolio as interactive game world
- Built with Phaser.js (WebGL), navigation through spatial exploration instead of menus
- "Press Start 2P" monospace font, 8-bit color palette with modern polish
- Demonstrates technical versatility through the portfolio's own architecture
- Noteworthy for the concept of portfolio-as-experience rather than portfolio-as-document

### Jasmine Gunarto — https://jasminegunarto.com/about/

- Indonesian designer (site blocked automated fetching — description based on user context)
- Relevant as a reference point for Indonesian design sensibility and visual language

### Kawe.ski — https://kawe.ski/

- **Minimalist modernism** — clarity, restraint, whitespace-first
- Senior product designer specializing in design systems (Nike, Hotmart, Banco Inter)
- Clean linear vertical flow, readability-first typography
- Neutral sophisticated palette, likely with dark/light mode
- The site practices what it preaches — scalable, documented design principles over flash

### Shruti Sonali — https://www.portfoliobyshruti.com/

- **Minimalist sophistication with warm, humanistic touch**
- Single-column, content-first layout optimized for readability
- Warm earthy tones (terracotta) — inviting rather than stark
- Unconventional personal touches (chai recipe section) that humanize the portfolio
- Built without Figma — no mockups, no handoff, AI-assisted workflow
- Multidisciplinary designer: brand identity, visual systems, UI

# Hello World: Building a Bold Creative Portfolio

The web is full of cookie-cutter portfolio sites. Same layouts, same colors, same everything. This project takes a different approach.

## The Philosophy

Bold design is not about being loud for the sake of it. It is about making intentional choices that create a memorable experience. Every gradient, every animation, every typographic decision serves a purpose.

## The Stack

We chose Astro for its performance-first approach. Combined with Tailwind CSS, we get a site that loads fast and looks incredible.

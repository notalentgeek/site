In this session we build a CLI tool from zero using Rust. We cover argument parsing with `clap`, async operations with `tokio`, and proper error handling.

### Topics Covered

- Project scaffolding with `cargo init`
- Argument parsing with `clap` derive macros
- Async file I/O with `tokio`
- Colored terminal output with `colored`
- Publishing to crates.io
